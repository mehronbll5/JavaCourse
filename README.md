# 📘 JavaCourse

Мой учебный репозиторий по изучению языка программирования Java с нуля — с целью подготовки к трудоустройству (Junior Java Developer).

Каждая тема курса оформлена в отдельном пакете с практическими задачами. По мере прохождения курса структура пополняется.

## 🗂 Структура проекта
src/main/java/org/example/
├── lesson01_basics/         — переменные, типы данных, вывод в консоль
├── lesson02_conditions/     — условия if/else, switch
├── lesson03_loops/          — циклы for, while, do-while
├── lesson04_arrays/         — массивы
├── lesson05_methods/        — методы (функции)
├── oop_basics/              — классы, объекты, наследование, полиморфизм
├── collections/             — List, Set, Map
└── ...

## 📌 О проекте
Курс проходится последовательно, тема за темой, с практическими заданиями после каждого урока. Цель — собрать портфолио для поиска работы Java-разработчиком.
